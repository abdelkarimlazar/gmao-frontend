# GMAO Frontend

Modern React frontend for a Smart Maintenance Management System (GMAO).

## Features

* Authentication and authorization
* Dashboard and statistics
* Equipment management
* Breakdown management
* Maintenance task management
* Notifications
* User management
* Intent assistant

## Technologies

* React
* JavaScript
* Axios
* React Router
* CSS

## Project Structure

```text
src/
├── api/
├── components/
├── context/
├── pages/
├── routes/
└── services/
```

## Backend Repository

https://github.com/abdelkarimlazar/gmao-backend

```bash
npm install
npm start
```
